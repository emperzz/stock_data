# AKShare 外汇 数据字典

共 **11** 个接口。

## 接口索引

- [`forex_spot_em`](./fx/forex_spot_em.md): 东方财富网-行情中心-外汇市场-所有汇率-实时行情数据
- [`forex_hist_em`](./fx/forex_hist_em.md): 东方财富网-行情中心-外汇市场-所有汇率-历史行情数据
- [`currency_boc_sina`](./fx/currency_boc_sina.md): 新浪财经-中行人民币牌价历史数据
- [`currency_boc_safe`](./fx/currency_boc_safe.md): 外汇管理局-人民币汇率中间价
- [`fx_spot_quote`](./fx/fx_spot_quote.md): 人民币外汇即期报价
- [`fx_swap_quote`](./fx/fx_swap_quote.md): 人民币外汇远掉报价
- [`fx_c_swap_cm`](./fx/fx_c_swap_cm.md): 中国外汇交易中心暨全国银行间同业拆借中心-基准-外汇市场-外汇掉期曲线-外汇掉期 C-Swap 定盘曲线
- [`fx_pair_quote`](./fx/fx_pair_quote.md): 外币对即期报价
- [`currency_pair_map`](./fx/currency_pair_map.md): 指定币种的所有能够获取到的货币对信息，历史数据可以调用 ak.currency_history() 获取
- [`macro_fx_sentiment`](./fx/macro_fx_sentiment.md): 货币对-投机情绪报告
- [`fx_quote_baidu`](./fx/fx_quote_baidu.md): 百度股市通-外汇-行情榜单
