# `energy_carbon_hb`

**描述**: 湖北碳排放权交易中心-碳排放权交易数据

**目标地址**: <http://www.cerx.cn/dailynewsOuter/index.htm>

**限量**: 返回从 2014-04-02 至今的所有历史数据


## 输入参数

| 名称 | 类型 | 描述 |
| --- | --- | --- |
| - | - | - |

## 输出参数

| 名称 | 类型 | 描述 |
| --- | --- | --- |
| 日期 | object | - |
| 成交价 | float64 | - |
| 成交量 | float64 | - |
| 最新 | float64 | - |
| 涨跌 | float64 | - |

## 接口示例

```python
import akshare as ak
energy_carbon_hb_df = ak.energy_carbon_hb()
print(energy_carbon_hb_df)
```
