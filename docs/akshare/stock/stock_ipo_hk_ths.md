# `stock_ipo_hk_ths`

**描述**: 同花顺-数据中心-新股申购与中签-港股

**目标地址**: <https://data.10jqka.com.cn/ipo/xgsgyzq/>

**限量**: 单次返回所有港股新股申购与中签数据

| - | - | - |

