# `news_cctv`

**描述**: 新闻联播文字稿, 数据区间从 20160330-至今

**目标地址**: <https://tv.cctv.com/lm/xwlb>

**限量**: 单次返回指定日期新闻联播文字稿数据


## 输入参数

| 名称 | 类型 | 描述 |
| --- | --- | --- |
| date | str | date="20240424"; 20160330-至今 |

## 输出参数

| 名称 | 类型 | 描述 |
| --- | --- | --- |
| date | object | 新闻日期 |
| title | object | 新闻标题 |
| content | object | 新闻内容 |

## 接口示例

```python
import akshare as ak
news_cctv_df = ak.news_cctv(date="20240424")
print(news_cctv_df)
```
