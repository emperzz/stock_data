# `air_city_table`

**描述**: 所有能获取空气质量数据的城市表

**目标地址**: <https://www.aqistudy.cn/>

**限量**: 单次返回所有可以获取的城市表数据


## 输入参数

| 名称 | 类型 | 描述 |
| --- | --- | --- |
| - | - | - |

## 输出参数

| 名称 | 类型 | 描述 |
| --- | --- | --- |
| 序号 | int64 | - |
| 省份 | object | - |
| 城市 | object | - |
| AQI | float64 | - |
| 空气质量 | object | - |
| PM2.5浓度 | object | - |
| 首要污染物 | object | - |

## 接口示例

```python
import akshare as ak
air_city_table_df = ak.air_city_table()
print(air_city_table_df)
```
