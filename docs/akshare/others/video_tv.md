# `video_tv`

**描述**: 艺恩-视频放映-电视剧集

**目标地址**: <https://www.endata.com.cn/Video/index.html>

**限量**: 返回前一日的电视剧播映数据


## 输入参数

| 名称 | 类型 | 描述 |
| --- | --- | --- |
| - | - | - |

## 输出参数

| 名称 | 类型 | 描述 |
| --- | --- | --- |
| 排序 | int64 | - |
| 名称 | object | - |
| 类型 | object | - |
| 播映指数 | float64 | - |
| 媒体热度 | float64 | - |
| 用户热度 | float64 | - |
| 好评度 | float64 | - |
| 观看度 | float64 | - |
| 统计日期 | float64 | - |

## 接口示例

```python
import akshare as ak
video_tv_df = ak.video_tv()
print(video_tv_df)
```
