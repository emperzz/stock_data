# `option_current_em`

**描述**: 东方财富网-行情中心-期权市场

**目标地址**: <https://quote.eastmoney.com/center/qqsc.html>

**限量**: 单次返回全部合约的实时行情


## 输入参数

| 名称 | 类型 | 描述 |
| --- | --- | --- |
| - | - | - |

## 输出参数

| 名称 | 类型 | 描述 |
| --- | --- | --- |
| 代码 | object | - |
| 名称 | object | - |
| 最新价 | float64 | - |
| 涨跌额 | float64 | - |
| 涨跌幅 | float64 | 注意单位: % |
| 成交量 | float64 | - |
| 成交额 | float64 | - |
| 持仓量 | float64 | - |
| 行权价 | float64 | - |
| 剩余日 | float64 | - |
| 日增 | float64 | - |
| 昨结 | float64 | - |
| 今开 | float64 | - |
| 市场标识 | int64 | - |

## 接口示例

```python
import akshare as ak
option_current_em_df = ak.option_current_em()
print(option_current_em_df)
```
