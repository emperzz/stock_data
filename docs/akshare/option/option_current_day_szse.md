# `option_current_day_szse`

**描述**: 深圳证券交易所-期权子网-行情数据-当日合约

**目标地址**: <https://www.sse.org.cn/option/quotation/contract/daycontract/index.html>

**限量**: 单次返回所有数据


## 输入参数

| 名称 | 类型 | 描述 |
| --- | --- | --- |
| - | - | - |
