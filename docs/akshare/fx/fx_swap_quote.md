# `fx_swap_quote`

**描述**: 人民币外汇远掉报价

**目标地址**: <http://www.chinamoney.com.cn/chinese/mkdatapfx/>

**限量**: 单次返回实时行情数据


## 输入参数

| 名称 | 类型 | 描述 |
| --- | --- | --- |
| - | - | - |

## 输出参数

| 名称 | 类型 | 描述 |
| --- | --- | --- |
| 货币对 | object | e.g., "USD/CNY" |
| 1周 | object | e.g., "11.50/12.00" |
| 1月 | object |  |
| 3月 | object |  |
| 6月 | object |  |
| 9月 | object |  |
| 1年 | object |  |

## 接口示例

```python
import akshare as ak
fx_swap_quote_df = ak.fx_swap_quote()
print(fx_swap_quote_df)
```
