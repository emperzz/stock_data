# `forex_spot_em`

**描述**: 东方财富网-行情中心-外汇市场-所有汇率-实时行情数据

**目标地址**: <https://quote.eastmoney.com/center/gridlist.html#forex_all>

**限量**: 单次返回所有实时行情数据
