**目标地址**: <https://www.soozhu.com/price/data/center/>

**限量**: 单次返回近半个月的历史数据

| 名称 | 类型 | 描述 |
| --- | --- | --- |
| - | - | - |

## 输出参数

| 名称 | 类型 | 描述 |
| --- | --- | --- |
| 日期 | object | - |
| 价格 | float64 | - |

## 接口示例

```python
import akshare as ak
```
