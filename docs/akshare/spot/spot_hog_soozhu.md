# `spot_hog_soozhu`

**目标地址**: <https://www.soozhu.com/price/data/center/>

**限量**: 单次返回所有实时数据


## 输入参数

| 名称 | 类型 | 描述 |
| --- | --- | --- |
| - | - | - |

## 输出参数

| 名称 | 类型 | 描述 |
| --- | --- | --- |
| 省份 | object | - |
| 价格 | float64 | - |
| 涨跌幅 | float64 | - |

## 接口示例

```python
import akshare as ak
spot_hog_soozhu_df = ak.spot_hog_soozhu()
```
